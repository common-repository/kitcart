== Description ==
Kitcart is a cloud-based ecommerce platform for online stores and retail point-of-sale systems that helps modern businesses sell anything to anyone anywhere.
The Kitcart plugin is a simple tool to keep track of sales of kitcart customers that keeps a woocommerce shop and report each order so as to keep inventory in sync.
